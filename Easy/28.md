<h1><a href="https://leetcode.com/problems/count-of-matches-in-tournament/description/">Count of Matches in Tournament</a></h1>

```cpp
int numberOfMatches(int n) {
        
        int ans=0;
        while(n!=1)
        {
            if(n%2==0)
            {
                ans+=n/2;
                n=n/2;
            }
            else
            {
                ans+=(n-1)/2;
                n=n/2 +1;
            }
        }
        return ans;
    }
```
